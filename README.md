# Labubu — сайт с вдохновением и оформлением заказов

## Демо

Посмотреть сайт можно по ссылке:https://nekkkkate.github.io/labubu-web/

## О проекте

Labubu — это небольшой многстраничный сайт о коллекционных игрушках-монстриках Labubu. 
Сайт объединяет:
- описание бренда и "вселенной" Labubu;
- каталог коллекций с возможностью добавить наборы в корзину;
- галерею вдохновения с милыми и забавными картинками;
- страницу с контактами и краткой философией бренда;
- заготовленные страницы аккаунта, корзины и отзывов.

Проект выполнен как учебная работа, с упором на аккуратную вёрстку, работу с DOM, эффектами интерфейса и хранением данных на стороне клиента.

## Основная функциональность

- **Главная страница** с кратким описанием бренда и призывом перейти к коллекциям или в галерею.
- **Каталог коллекций игрушек** с несколькими тематическими наборами ("Волшебный лес", "Космические исследователи" и др.).
  - Выбор количества наборов перед добавлением.
  - Добавление выбранных коллекций в корзину с сохранением данных в `localStorage`.
  - Визуальная обратная связь (уведомления и анимация кнопки).
- **Галерея вдохновения** с возможностью открыть изображение в полноэкранном модальном окне.
  - Переключение картинок стрелками на клавиатуре.
  - Счётчик текущего изображения.
- **Страница контактов/о проекте** с:
  - контактами (почта, телефон, мессенджеры, адрес в Санкт‑Петербурге);
  - описанием философии бренда;
  - небольшим прайс‑листом коллекций.
- **Общие UI‑функции**:
  - подсветка активного пункта меню в навигации;
  - вывод времени загрузки страницы в подвале;
  - переиспользуемые уведомления и диалог подтверждения.

## Структура страниц

1. **index.html** — главная страница.
2. **collections.html** — каталог коллекций игрушек и добавление в корзину.
3. **gallery.html** — галерея вдохновляющих изображений с модальным окном.
4. **about.html** — контакты, информация о бренде, прайс‑лист.
5. **account.html** — страница аккаунта пользователя (заготовка под личный кабинет).
6. **cart.html** — корзина товаров (работа с данными из `localStorage`).
7. **reviews.html** — страница для отзывов.

## Используемые технологии

- **HTML5** — семантическая разметка и структура страниц.
- **CSS3** — оформление, переиспользуемые компоненты (`css/style.css`, `css/components.css`, `css/pages.css`).
- **Vanilla JavaScript (ES6+)** — логика интерфейса без сторонних фреймворков:
  - работа с `localStorage` для хранения корзины;
  - модальные окна и уведомления;
  - обработка событий клавиатуры и кликов;
  - измерение и показ времени загрузки страницы.
- **GitHub Pages** — деплой статического сайта.

## Как запустить проект локально

1. Склонировать репозиторий или скачать архив проекта.
2. Открыть файл `index.html` в любом современном браузере.
3. Для более корректной работы (как на продакшене) можно запустить простой локальный веб‑сервер и открыть корень проекта в нём.

Проект можно использовать как основу для дальнейшего развития: добавления форм авторизации, более сложной логики оформления заказов или панели администратора для управления коллекциями.
